IrisGuard – Eye Protection SystemIrisGuard is a sophisticated desktop utility developed in Java designed to mitigate the risks of Computer Vision Syndrome (CVS). Unlike traditional reminder apps that are easily ignored, IrisGuard follows a "strict enforcement" philosophy to ensure users actually take the breaks they need.👁️ The 20-20-20 RuleInspired by the 20-20-20 rule, the application creates a mandatory physical and digital boundary by requiring users to look at something 20 feet away for 20 seconds every 20 minutes.✨ Key FeaturesIntrusive Break Overlay: Upon timer completion, a full-screen JDialog is launched that stays "Always on Top," effectively locking the user out of their work environment to ensure compliance.Intelligent Anti-Bypass: Includes movement-sensing logic that detects if a user is still interacting with the computer during a break. If activity is detected, the timer resets.Dynamic Health Visualization: A JProgressBar acts as a "Health Bar," providing a visual representation of the user's remaining "eye stamina".Sophisticated State Management: Internal logic ensures that "Focus" and "Break" states do not overlap or conflict.🛠️ Technical ImplementationThe system is built using Java Swing and follows core Object-Oriented Programming (OOP) principles.Workflow & AlgorithmInitialization: Sets global constants (Focus = 1200s, Break = 20s).Focus Monitoring: A javax.swing.Timer decrements the focus clock every second.The Intervention: When the clock hits zero, startStrictBreak() initiates.Verification: A MouseMotionListener tracks cursor movement. If movement exceeds a 5-pixel threshold, the break timer resets to 20 seconds.Recovery: Once the break completes without interruption, the overlay is disposed of, and the cycle repeats.🚀 Future RoadmapComputer Vision Integration: Replacing mouse detection with webcam-based blink and gaze tracking.Administrative Reporting: Logging "Health Marks" to create weekly reports on break compliance.System-Level Hooks: Disabling keyboard shortcuts (like ALT + Tab) during breaks to prevent bypassing the system.Cloud Synchronization: Syncing eye-health profiles across multiple devices.👤 AuthorNikhil
